Python worst Practices AKA Anti-Patterns

Abstract: This talk is about the common Anti-Patterns in Python. Among overwhelming resources to teach best practices in Python, it is more scary that there are very few places which will teach you worse practices. There are plenty of ways to write bad code. But in Python, one in particular reigns as king.

About the speaker: Pratibha Jagnere is a Sr Python developer of her company looking over the development and maintenance of backend including APIs and internal python packages. Via this talk, she wants to share how they improved their code by removing anti-patterns.
